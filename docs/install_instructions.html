Factium AI Distribution Hub

Follow these steps to package this app for any device for FREE without needing Node.js or admin rights.

1. Hosting the PWA

Your current AI Studio environment provides a live URL. Copy that URL (e.g., https://factium-ai.aistudio.com). All converters below will use this link.

2. Windows .EXE (PWA2EXE)

  1. Go to PWA2EXE.com
  2. Paste your app URL.
  3. Select "Windows".
  4. Click "Generate" and download the setup.exe.

3. Android .APK (PWA2APK)

  1. Go to PWA2APK.com
  2. Enter your URL.
  3. Fill in "Factium AI" as the app name.
  4. Wait for the build to finish and download the .apk to your phone.

4. macOS / iOS

iOS: Open the URL in Safari on your iPhone, tap "Share", then "Add to Home Screen". It will work exactly like an installed app.

macOS: Open the URL in Chrome, click the "Install" icon in the address bar (the square with an arrow).

5. Offline Use

Once you open the app for the first time on any device, the Service Worker will automatically cache all files. You can now close your internet and it will still open!
